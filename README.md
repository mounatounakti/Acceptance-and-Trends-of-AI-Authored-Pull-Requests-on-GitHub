### Project Goal
The goal of this project is to analyze the impact and adoption of AI-generated contributions in open-source software development by examining AI-authored pull requests on GitHub.

### Requirements
- Python 3.9 or higher
- Internet connection (to load the dataset from Hugging Face)

### Dataset
The study uses the **Hugging Face release of the AIDev dataset**
https://huggingface.co/datasets/hao-li/AIDev

### Install Dependencies
Run the following command:

```bash
pip install pandas datasets numpy matplotlib scikit-learn
```

### To ensure reproducibility:
- Install the required dependencies  
- Run the notebooks in sequence  
- Use the provided dataset
